# WriteHere Front
React + Vite로 구성되어 있습니다.
## 설치된 라이브러리
- react-router-dom
- tailwindcss (v3)
- react-responsive
- jwt-decode
- zustand (Redux 대체)
- validator (간단 유효성 검증)
- dayjs
- axios (좀 더 편리한 fetch를 위해)
- tiptap (에디터)
- shadcn (컴포넌트 공통화)
- lucide (아이콘 내장화)

## 환경 설정 파일
### .env.development
요청 주소가 하나 있으며 게이트웨이를 사용하지 않을 경우 수정

### .env.production
게이트웨이 방식으로 사용됨
